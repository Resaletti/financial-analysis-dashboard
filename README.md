# 💰 Financial Analysis Dashboard

An interactive Power BI dashboard designed to analyze financial performance, revenue streams, expenses, and profitability metrics.

---

# 🚀 Project Overview

This project provides financial insights through interactive visualizations and KPIs, helping businesses understand revenue composition, expense distribution, and profit margins.

---

# 📊 Dashboard Preview

![Dashboard](https://raw.githubusercontent.com/Resaletti/financial-analysis-dashboard/main/images/dashboard-preview.png)

---

# 🔍 Key KPIs

- Total Revenue
- Total Expenses
- Profit Margin
- Revenue by Component
- Expenses by Component
- Financial Segmentation Analysis

---

# 📈 Key Insights

- Total revenue reached 1.92M
- Profit margin achieved 39.96%
- Sales represented the largest revenue source
- Administrative and technology costs were the highest expenses
